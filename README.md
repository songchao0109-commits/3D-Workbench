# 3D Workbench

轻量级 3D 分镜预演工作台，用于快速产出带有运镜和空间关系的参考视频，为 AI 生视频提供更可控的输入参考。

## 项目定位

3D Workbench 不是 Blender、Maya 或 Unreal 的替代品，也不是完整的 DCC 动画软件。

它更像一个为 AI 视频前置预演设计的轻量虚拟摄影台，让创作者先搭建场景、安排角色与镜头运动，再输出能表达构图、机位和空间关系的参考内容，作为后续 AI 生视频的输入依据。

## 当前已实现能力

- 支持导入 `.glb` 模型
- 3D 场景浏览、对象点选与视口拖拽编辑
- 对象移动、旋转、缩放、隐藏、锁定、删除
- 材质颜色、粗糙度、金属度、透明度与贴图替换
- 摄影机创建、视角切换、FOV 调整、自由朝向与注视目标
- 世界属性编辑，包括地面、网格和全景图背景
- 带骨骼模型的 `FK / IK` 静态 Pose 调整
- 快照工作区与参考帧导出
- 常用画幅比例切换
- 关键帧时间线第一版
- 对象、机位与骨骼控制的关键帧记录与播放预览
- 机位片段轨与基础镜头切换预演

## 适合谁使用

- 导演 / 分镜创作者：快速验证镜头和构图
- AI 视频创作者：在生成前锁定空间关系、机位和运镜参考
- 预演执行 / 3D 协作人员：快速输出可沟通的镜头参考

## 使用场景

- 导入角色、道具、环境模型，快速搭建场景
- 调整人物站位、道具关系和镜头构图
- 为角色制作静态骨骼 Pose 或基础关键帧动作
- 创建多个机位并规划镜头切换
- 导出参考帧与动画预演结果，用于后续 AI 视频生成参考

## 技术栈

- Vite
- React 19
- TypeScript
- Three.js
- Zustand

## 本地运行

```bash
npm install
npm run dev
```

## 当前边界

当前版本聚焦轻量预演闭环，暂不包含：

- 完整 DCC 级动画制作流程
- 多人协作
- 本地项目持久化或工程文件读写
- 真实视频渲染导出
- AI 视频平台接口直连
- 多格式模型导入（当前仅支持 `.glb`）
